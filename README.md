# Objectives
- Tahap 1: Membangun REPL (Read-Evaluate-Print Loop):Target: CLI bisa menerima input terus-menerus
- Tahap 2: Parsing Perintah (Command Tokenization):Target: Memisahkan perintah utama dan argumen
- Tahap 3: Implementasi Built-in Commands:Target: Perintah yang dieksekusi langsung oleh shell (tanpa fork)
- Tahap 4: Implementasi Forking & Eksekusi Perintah Eksternal:Target: Bisa menjalankan perintah bawaan OS (ls, mkdir, clear, dll)
- Tahap 5: Implementasi Fitur Advanced (Piping / Redirection)
- Tahap 6: Pengujian Beban & Penanganan Eror:Target: Memastikan CLI tidak crash saat salah ketik
